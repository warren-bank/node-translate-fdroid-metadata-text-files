叫我伊什梅尔

<ul>
<li>几年前 -- 我从未想过，我的钱包里有多少钱，我的钱包里没有钱，也没有什么特别让我感兴趣，我想我要开个小船，看看世界上的水。</li>
<li>这是我驱离脾脏，调节血液循环的方法。</li>
<li>每当我发现自己越来越冷酷的嘴巴时，只要是潮湿的，11月在我灵魂里的灰熊，每当我发现自己不自觉地在棺木仓库前暂停使用，并在每次我的葬礼结束后，我都会迂到 ; 特别是当我的伪君子有我的上风时，它需要一个强烈的道德原则来阻止我故意进入大街上，并且有条不紊地把人们的帽子敲开 -- 然后，我就会在我能做到的时候，尽快地去海边。<ul>
<li>这是我的手枪和球的替代品</li>
<li>带着一个哲学繁荣的凯托把自己扔在了他的剑上 ; 我静悄悄地去了船。</li>
<li>这没什么好奇怪的 如果他们知道，几乎所有在学位，时间或其他方面的男性，都对与我的海洋有非常接近的感受。</li>
</ul>
</li>
</ul>
